# uptime 
- monitor the cpu load average
- shows for 1 minute, 5 minute and 15 minutes
- linux includes network and disk operations, as high usage of these can lead to CPU load
```
[root@atollon-client ~]# uptime
 02:10:21 up 22:34,  4 users,  load average: 0.07, 0.05, 0.01
```

# nproc 
- number of CPU
- 8 cpu 
```
[root@atollon-client ~]# nproc
8
```

# If load average is less then 8.00 , then cpus are under loaded
# If load average is == 8.00, then cpus are busy
# If load average is > 8.00, then cpus are busy and other processes are waiting for the cpu time.
