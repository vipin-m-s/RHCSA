# Configuring DNF repositories
## 1. Manual configuration 
- Adding .repo file in /etc/yum.repos.d
- Adding the [repository] in /etc/dnf/dnf.conf
- .repo takes precedence and recommended to use .repo
- .repo file contains the repo link, if gpg key is enabled then the link to gpg key,

## 2. command line configuration 
- Adding via dnf config-manager
```
dnf config-manager \
--add-repo="https://dl.fedoraproject.org/pub/epel/10/Everything/x86_64/"
```
