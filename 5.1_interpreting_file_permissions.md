# There are 3 permissions
- owning user
- Owning group
- others ( any user other then owning user or not part of owning group ) 


# Permissions
- 4 read
- 2 write
- 1 execute

# File permissions
- To view the file - read
- To write to the file - write
- If it has executable code and to execute - execute

# Dir permisssions
- To cd into the directory - execute
- To list contents in the directory - read and execute
- TO create/delete file within the directory - write and execute
