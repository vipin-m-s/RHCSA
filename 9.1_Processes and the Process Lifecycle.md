# We can use ps tree
- systemd is ultimate parent process with process id of 1
- Zombie process - process who has released all  his reosurces except his PID
- Fork - create a new process by forking its parents address space

# top command 
- real time updates about a process
- Shift + p - sort based on cpu
- shift + m - sort based on memory

# job control 

## & - send process to the background 
- terminal is occupied
- we cannot run any command
```
[root@atollon-client ~]# sleep 10
```
- run process in background
```
[root@atollon-client ~]# sleep 1m &
[1] 6854
Thu Apr 16 01:41:46 AM UTC 2026
```

## ctrl + z - Stop a proccess
```
[root@atollon-client ~]# sleep 5m
^Z
[1]+  Stopped                 sleep 5m
```

```
[root@atollon-client ~]# jobs
[1]+  Stopped                 sleep 5m
```

- bring job to foreground
```
[root@atollon-client ~]# fg %1
sleep 5m
```

- send job to background
```
[root@atollon-client ~]# bg %1
[1]+ sleep 5m &
```
