# creating file
- we can use cat echo to redirect to file and add contents
- We can use text edutor to create file
- We want empty file for later use. We can use touch command
- touch file1
- If file already exists, timestamp is updated

# Creating multiple files
- touch file1{1..100}.htm

# Creating folder
- mkdir dir1
- mkdir dir1  dir2 dir3
- mkdir -pv /dev/test/new
  - -p (parent) Creates directory recursively
  - -v (verbose)

# copying contents
- cp file1 dir1
- cp file1 f2 dir1
- ls dir1 dir2

# removing diles and dirs
- rm file1
- rm dir1 >>> fails telling dir
- rm -r dir1 >>> asks for prompt for all dirs
- rm -rf dir1 >>> delete without prompt

- rm -i dir1 ---> interactive prompt before deletintg
- rmdir can be used to remove empty directory
- To remove dir with contents we should use rm -r dir1
