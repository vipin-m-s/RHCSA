# locate command
- uses locate database which has reference to files
- New files will not be located
- WE need to use `updatedb` command to re-initialize the database

# we can perform acting on found files
- does not ask confirmation 
```
[root@ceph-vipin-automation-55oz7m-node7 ~]# find /tmp -iname "*.htm" -delete
[root@ceph-vipin-automation-55oz7m-node7 ~]#
```

# If we want confirmation 
```
[root@ceph-vipin-automation-55oz7m-node7 ~]# find /tmp -iname "*.htm" -exec rm -i {} \;
```
