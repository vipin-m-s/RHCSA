- command line is text based interface for executing commands
- command line interface is provided by program called shell
- Default shell in RHEL is GNU Bourne Again SHell BASH
- Regular user - $
- Root user - #
- ssh is provided by OpenSSH program
- We can use password, private+public key for authentication
- When we connect with remote host, the remote host gives its host fingertprint key to the client and client saves it.
- If someone tries to impersonate the host, The other host will have different key and hence we will see warning and connection is closed.

# command line shortcuts
- ctrl + A - go to begiing
- ctrl + E - go to end
- cltrl + u - clear from cursor to beginning
- ctrl + k - delete from cursor to end
- ctrl + < - jump words
- ctrl + > - jump words right
- ctrl + R - reverse search

# unconditional command excution 
- doesnt matter if foo or bar fails, next command runs. 
```
[root@ceph-vipin-tfa-q5w08s-node6 ~]# whoami ; foo; bar; pwd
root
-bash: foo: command not found
-bash: bar: command not found
/root
```

# conditional command execution 
- whoami passed hence move to foo
- foo failed stop the chain of execution 
```
[root@ceph-vipin-tfa-q5w08s-node6 ~]# whoami && foo && bar && pwd
root
-bash: foo: command not found
```

# double Pipes
- If the command before pipe is error then only move next
- If the command before pipe is not error stop the chain
```
[root@ceph-vipin-tfa-q5w08s-node6 ~]# whoami || date || foo || bar || pwd
root
```




```
[root@ceph-vipin-tfa-q5w08s-node6 ~]# foo || bar || whoami
-bash: foo: command not found
-bash: bar: command not found
root
```

foo failed -> bar failed -> whoami success ( stop the chain !!!! ) 
```
[root@ceph-vipin-tfa-q5w08s-node6 ~]# foo || bar || whoami  || date
-bash: foo: command not found
-bash: bar: command not found
root
```

- whoami - pass
- && foo - execute foo only if previous commaand pass ( execute and fails ) 
- || bar - execute bar only if previous command fails ( execute ) 
```
[root@ceph-vipin-tfa-q5w08s-node6 ~]# whoami && foo || bar
root
-bash: foo: command not found
-bash: bar: command not found

[root@ceph-vipin-tfa-q5w08s-node6 ~]# whoami && foo || date
root
-bash: foo: command not found
Wed Apr 15 03:20:44 UTC 2026
```
