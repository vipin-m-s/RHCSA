# Flatpak 
- There will flatpak repo1 , flatpak repo2
- We can install app1 from repo1 and app2 from repo2
- app1 and app2 wwill be installed and run in sandbox
- one app cannot access another app
- flatpak runtime - contains system level libraries whcih flatpak can share a flatpak runtime
- 

# install flatpak 
```
dnf install flatpak
podman login registry.redhat.io
```

# list remote repository 
flatpak remotes

# List apps in remote repository
flatpak remote-ls --app 

# Add a remote repository 
flatpak remote-add --if-not-exists fedora oci+https://registry.fedoraproject.org
flatpak remotes

# Removing a remote repositotyr 
flatpak remote-delete fedora

# disable 
flatpak remote-modify --disable fedora

# enable 
flatpak remote-modify --enable fedora
